BotBuilders Academy - Website Plan

1. Site Name

BotBuilders Academy

Why this name? The name reflects the purpose of the site: teaching children how to build and program robots in a fun and interactive way.

2. Site Purpose

BotBuilders Academy is an online platform offering robotics and coding courses for kids. It provides beginner-friendly lessons, hands-on activities, and interactive events.

3. Scenarios

Visitors to the site may ask:

  • How can my child start learning robotics? The site will offer beginner courses that introduce fundamental concepts.
  • Are there any interactive activities? The site includes hands-on projects, coding challenges, and virtual robotics simulations.

4. Color Schema

The website will use the following colors:

  • Blue (#007BFF): Used for headings and buttons to create an engaging, tech-inspired feel.
  • Yellow (#FFC107): Used for accents and highlights to make the site visually appealing to kids.

5. Typography

The fonts selected for the site are:

  • Poppins (Headings): A modern and playful font that appeals to kids.
  • Open Sans (Body Text): A clean and readable font for paragraphs and instructions.
